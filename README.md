# To-Do List App

A simple and clean To-Do List web application built with vanilla HTML, CSS, and JavaScript.

## Features

- Add tasks by typing and pressing Enter or clicking the Add button
- Mark tasks as complete with a single click
- Delete individual tasks
- Filter tasks by: All, Active, or Done
- Clear all completed tasks at once
- Tasks are saved in the browser (localStorage) — they stay even after refreshing

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

1. Download or clone this repository
2. Open `index.html` in any web browser
3. No installation or setup required!

## How to Use

1. Type a task in the input box
2. Press **Enter** or click **+ Add**
3. Click the circle button to mark a task complete
4. Click **✕** to delete a task
5. Use the filter buttons (All / Active / Done) to view tasks
6. Click **Clear completed** to remove all finished tasks
